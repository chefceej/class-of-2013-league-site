Week Config Admin

Map ESPN weeks to matchup weeks

ESPN sometimes creates short partial weeks that the league combines into one matchup week. Use the Matchup Week inputs below to assign each ESPN week to the correct matchup week. To merge two ESPN weeks (e.g. weeks 3 and 4) into one matchup week, set both to the same number. Weeks must be assigned in non-decreasing order.

Workflow After Editing

  1. Click Download week_config.json
  2. Move the downloaded file to docs/data/week_config.json in your local repo
  3. Run python3 src/fetch_data.py locally to regenerate league data
  4. Run git add docs/data/ && git commit -m "chore: update week config" && git push
